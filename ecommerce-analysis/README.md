# E-commerce Analysis

## Project Overview
This project focuses on analyzing e-commerce sales data using SQL to generate business insights related to customer behavior, product performance, and revenue trends.

## Objectives
- Analyze total sales revenue
- Identify top-selling products
- Find high-value customers
- Track monthly sales trends
- Evaluate category performance

## Dataset Information
The dataset contains customer orders, products, sales transactions, and revenue-related information.

## SQL Concepts Used
- SELECT Statements
- WHERE Clause
- GROUP BY
- ORDER BY
- Aggregate Functions
- INNER JOIN
- LEFT JOIN
- Subqueries
- Common Table Expressions (CTEs)
- Window Functions

## Tools Used
- MySQL
- PostgreSQL
- GitHub

## Key Insights
- Electronics category generated the highest revenue.
- Returning customers contributed significantly to sales.
- Monthly sales increased during festive seasons.
